# driver-discovery

## Setup Instructions


### Backend:
create .env file as specified in   `./backend/example.envfile` as `dev.env`

```
cd backend
npm install
npm run dev
```

for test:
```
npm run test
```

for production
create new envfile production.env
```
npm start
```
